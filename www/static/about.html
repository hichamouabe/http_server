About This Server

This is a custom HTTP/1.1 server written in C++98.

Built from scratch — no external libraries.

  • Non-blocking I/O with epoll
  • HTTP/1.1 keep-alive
  • Static file serving
  • File uploads via POST
  • File deletion via DELETE
  • Directory autoindex
  • Custom error pages
  • Redirects
  • Virtual hosting
