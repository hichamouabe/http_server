Webserv/1.0 — Test Suite

Server is running. Use the links below to test each feature.

GET — Static Files

  • GET /index.html — this page (200 OK)
  • GET /about.html — another static page (200 OK)
  • GET /style.css — CSS file (200 OK)
  • GET /sample.txt — plain text file (200 OK)

GET — Directory Listing (Autoindex)

  • GET /images/ — directory listing (200 OK)

GET — Error Pages

  • GET /nonexistent.html — custom 404 page
  • GET /errors/403.html — custom 403 page

GET — Redirects

  • GET /redirect — 301 redirect to /
  • GET /old-page — 301 redirect to /index.html

POST — File Upload


Or test with curl:

curl -X POST http://localhost:8080/upload --data "hello world"

DELETE — File Deletion

First create a file, then delete it:

curl -X DELETE http://localhost:8080/files/sample.txt

Curl Test Commands

curl -v http://localhost:8080/

curl -v http://localhost:8080/nonexistent

curl -X POST http://localhost:8080/upload --data-binary @yourfile.txt

curl -X DELETE http://localhost:8080/files/sample.txt

curl -X POST http://localhost:8080/readonly --data "test" (expect 405)

curl -v -L http://localhost:8080/redirect (follow redirect)
